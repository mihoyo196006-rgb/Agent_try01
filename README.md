# Morning Feishu Agent

This repo sends one compact Feishu bot message every morning.

## Current Path

Dayflow input comes only from Cloudflare Worker/KV:

1. Dayflow desktop saves state.
2. `D:\try\cloud-sync.js` posts a compact snapshot to the Worker.
3. GitHub Actions reads the Worker endpoint at 09:00 Beijing time.
4. `send_morning_plan.py` sends the Feishu message and records a delivery marker.

The previous Windows scheduled local snapshot read, commit, and push path has been removed.

## Token And Quota Use

The default message is generated by deterministic Python templates. It does not call an LLM and spends 0 model tokens.

GitHub Actions runs once per day by schedule. A scheduled run performs one Cloudflare Worker read, one Feishu tenant-token request, and one Feishu message send. Dayflow desktop writes are debounced and unchanged snapshots are skipped.
